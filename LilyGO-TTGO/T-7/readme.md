## Product Description

| Hardware                | Specifications |
| :-                      | :- |
| Main chip               | ESP 32 |
| Flash memory chip       | 25Q32 |
| Charging IC chip        | TP4054 |
| USB to serial port chip | CH340C |
| JST Connector           | 2Pin 1.25mm |
